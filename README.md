# OLX parser

Script for parsing olx.kz

## Installation

Use the git clone for install

```bash
git clone https://github.com/DiZiNnEs/OLX-parser
git cd OLX-parser
pipenv install
python3 runner.py
```

## Usage

```bash
python3 parser.py
```
